HTML Markup

Consists of an opening tag, a closing tag, and contents.
Examples: <b>Hello</b> will bold the word Hello <em>Hi</em> will italicize or place emphasis on the word Hi

HTML Attributes

Attributes are assigned using the following two tags:
<tag attr="value"> CONTENTS </tag>,
<a href="value"> CONTENTS </a> will add a hyperlink to the word in CONTENTS.
<img src="url" alt="text"> will add an image from the source URL.

Inline and Block

  • <br> - ends line (inline)
  • Block will create a "block" which has height and width
  • Block has an opening and closing tag
  • Example: <p> and </p>

Lesson Two: Creating a Structured Document

HTML, CSS, and Javascript; components of a house:

  • HTML is like the structure
  • CSS is like the style
  • Javascript is like the interactive elements

Tree-Like Structure

Webpages have a tree like structure with elements nested inside.
Elements form nested boxes and comprise the overall structure of any webpage.
Boxes, defined by the <div> tag can be given "attributes".
Attributes provide a description of what is contained in that section.

Extremely Important!

  • Use HTML and CSS Validator
  • Watch the container <div> and <span> elements
  • Use structured documents
  • Avoid Repetition
  • Avoid repetition for two reasons:
  • Typing the same or similar code repeatedly allows more possibility for errors. If the HTML is complex this may cause a major issue when debugging. Even copy/paste could present concerns when changes are needed.
  • Ultimately, nobody has the time to write as much code as is typically required. The CSS in this code uses no more than nine functions for the entire document.
